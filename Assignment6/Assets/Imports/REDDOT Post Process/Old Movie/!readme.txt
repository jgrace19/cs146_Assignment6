Old Movie Post Process 1.0
--------------------
Old Movie Post Process is postprocess effect that simulate old movie. Full source code provided : shaders and scripts.
Works on DX11 DX9 OpenGL. Require Unity Pro and SM3.0

Features
--------
- Fully random generated - all inside Shader!
- 1 Draw Call
- 3 modes (Simple, Medium, Full)
- Works in Color or Black and White
- Color Tint option
- Grain Intensity
- Dirt Size
- Frequence Scale (works independent of game speed)

Usage 
-----
Simply drag script to your camera. Default settings will be provided. You can play with it to achieve desire effects.
